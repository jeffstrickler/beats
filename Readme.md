Beats
=====

What is this all about?
-----

Beats are the time unit of Swatch Internet Time, a decimal time concept created by Swatch. In Swatch Internet Time the day is divided into 1000 beats, using Biel Meantime (equivalent to Central European time) as its time scale.  

This project was written in an evening as a simple Node app example. 


Installation
-----

- Run `npm install` 
- Run `node server.js`
- Point browser at `http://localhost:3000`
